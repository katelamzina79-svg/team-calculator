# Командный калькулятор

## О проекте
Это учебный проект для освоения командной работы с Git. Мы создали простой калькулятор с четырьмя базовыми операциями, чтобы на практике разобраться с ветками, пул-реквестами и тегами.

## Как запустить
```bash
# 1. Клонируем репозиторий
git clone https://github.com/katelamzina79-svg/team-calculator.git
cd team-calculator

# 2. Устанавливаем зависимости
pip install -r requirements.txt

# 3. Запускаем
python main.py
